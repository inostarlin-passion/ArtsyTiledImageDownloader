# 1 项目介绍

本工具支持知名艺术网站Artsy的tile化原图的解析和下载。

# 2 使用方法

首先安装依赖：
```bash
pip3 install -r requirements.txt
```

下载图片，执行以下命令（默认下载到 `~/Downloads/` 目录）：
```bash
./main.py [url]
```

示例：
```bash
./main.py https://www.artsy.net/artwork/nick-doyle-bathing-in-your-cold-light
```
